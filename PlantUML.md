# PlantUML 用脚本画UML图

## VS Code 配置plantuml
下载plantuml.jar文件
配置环境变量 PLANTUML_JAR 指向 plantuml.jar  
在VS Code 里安装插件okazuki.okazukiplantuml

##如果使用 
ctrl + shift + p 启动命令 输入 uml p 启动PlantUML Preview  
左边输入脚本，右边将实时显示UML图像
